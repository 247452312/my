前台请求值和后台返回值的标准


前台请求:
{
	"token":"xxxxxxxxxxxxxxxxxx",
    "interfaceName": "XxxService",
    "methodName": "xxXxxXxx",
    "args": {
    /* 需要的参数 */
    }
}

后台返回:(后台返回分为三种,A成功,B客户端传值错误,C服务器错误)
{
    "token": "xxxxxxxxxxxxxxxxxx",
    "code": 200, // 200代表请求成功,400代表传值错误,500代表服务器错误
    "msg":"",
    "data":{
        "status":true
    }
}
